# 净土法音 - 设计上下文 (Design Context)

> 此文件为 Impeccable / frontend-design 技能提供项目设计上下文。  
> 进行 UI/UX 设计或重构时，Claude 会优先读取此文件。

## Target Audience（目标用户）

- **主要用户**：佛教净土宗修行者、对佛法感兴趣的学习者
- **使用场景**：日常听经、念佛、阅读讲记、移动端通勤收听
- **年龄分布**：中老年为主，也有年轻学佛者
- **技术能力**：部分用户不熟悉复杂交互，需要简洁直观

## Use Cases（核心用例）

1. **听经台**：浏览、播放大安法师讲经音频，支持后台播放、进度记忆
2. **有声书**：收听印光大师永怀录、净土圣贤录等有声书
3. **法音文库**：在线阅读讲记文稿，支持搜索、目录导航
4. **佛号念诵**：循环播放佛号音频，专注修行
5. **留言墙**：用户留言、祈福、分享心得

## Brand Personality / Tone（品牌调性）

- **宁静庄严**：佛教内容需体现恭敬、清净
- **温和克制**：避免花哨动效，保持禅意与克制
- **传统与现代融合**：既有古典书法/纹样元素，又符合现代阅读习惯
- **可访问性优先**：字体清晰、对比度足够，适合长时间阅读/收听

## Technical Constraints（技术约束）

- 纯前端：HTML/CSS/JS，Vite 构建
- 无框架：无 React/Vue，使用原生或轻量方案
- 移动优先：PWA，支持离线、添加到主屏幕
- 多语言：zh-CN 为主，支持 en、fr

## Differentiation（差异化）

- **净土法音** 应让人一眼感受到「清净」与「法喜」
- 避免通用 SaaS 风格，避免科技感过强的冷色调
- 可参考：古籍排版、禅意留白、暖色纸质感、适度传统纹样
