## 设计灵魂

```
宣纸为底，墨色为骨，朱砂为魂，留白为禅
```

**核心原则**：不是"加水墨效果"，而是"用水墨语言表达界面"。

## 水墨 UI 语言映射

| 水墨概念 | UI 对应 | CSS 实现 |
|---------|--------|---------|
| **宣纸** | 背景色 + 极淡纸纹 | `--bg: #F7F5F0` + CSS noise overlay `opacity: 0.03` |
| **浓墨** | 标题文字 | `color: #2D2824` |
| **淡墨** | 次要文字 | `color: rgba(45,40,36, .45)` |
| **飞白** | 分割线 | `linear-gradient(to right, transparent 5%, rgba(45,40,36,.12) 30%, rgba(45,40,36,.12) 70%, transparent 95%)` |
| **朱砂印** | 播放按钮、AI 标记 | `color: #C04B2D` / 小方印章风格 |
| **留白** | 大间距 | section 间距 56-64px |
| **一笔一意** | 最少元素 | 去掉冗余装饰、边框、阴影 |

## 色彩系统

| 角色 | 当前 | 水墨版 |
|------|------|--------|
| 主背景 | `#F9F8F6` | `#F7F5F0`（更暖，宣纸） |
| 强调色 | `#D97757` 赤陶 | `#C04B2D` 朱砂红 |
| 浓墨（标题） | `#1E1E1E` | `#2D2824` 焦墨 |
| 淡墨（正文） | `rgba(30,30,30,.6)` | `rgba(45,40,36,.5)` |
| 极淡墨 | `rgba(30,30,30,.35)` | `rgba(45,40,36,.25)` |
| 暗模式底 | `#1D1D1D` | `#1A1814` 墨宣纸 |
| 暗模式墨 | `#E8E0D5` | `#E5DDD2` |

## 排版系统

- **法语区**：Noto Serif SC, 1.25rem, line-height 2.0, 居中
- **section 标题**：Noto Sans SC, 0.75rem, 淡墨, letter-spacing 0.08em
- **内容文字**：Noto Sans SC, 0.9rem, 浓墨

## 关键变更点（实施时参考）

### 需要修改的文件
1. `src/css/tokens.css` — 色彩系统更新（水墨色板）
2. `src/css/pages.css` — 首页样式重写（去卡片化）
3. `src/js/pages-home.js` — HTML 结构简化

### 具体 CSS 变更
1. 去掉 `.home-*-card` 的 `background`, `border-radius`, `box-shadow`, `::before` 伪元素
2. 法语区改用 `font-family: var(--font-serif)`, 加大字号和留白
3. 添加飞白分隔线样式（CSS 渐变）
4. AI badge 改为印章风格（小方块 + 红底白字）
5. 佛号区改为纯文字横排（去掉播放按钮圆）

### 不变的部分
- JS 逻辑（事件处理、数据加载）基本不变
- 播放功能不受影响
- 暗色模式需同步调整（浓墨→淡宣纸底）

## 参考图分析

两张参考图的共同特征：
- 近乎纯白的宣纸底色
- 大面积留白（70%+ 的画面是空白）
- 中文经文 + 梵文/英文双语
- 水彩/水墨画法，柔和边缘
- 色彩极度克制（只有画面主体有颜色）
- **关键：安静到让人想闭眼念佛**

## 零动画原则

- 无 `animation`、无 `@keyframes`
- 仅保留最基础的 `transition: opacity 0.15s` 用于切换
- 骨架屏改为简单的淡灰横线（不脉冲）
- hover 状态只做微弱 opacity 变化
