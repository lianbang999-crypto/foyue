# 净土法音 Figma 智能体创作 Brief

## 1. 项目定位
请为“净土法音”设计一套移动端优先的网站 UI/UX。

这不是资讯站，也不是音乐播放器，而是一个帮助用户长期听经、查原文、引用问法的修学入口。

核心关键词：
- 极简
- 静修书卷
- 温暖纸本
- 低干扰
- 可沉浸
- 可长期使用

## 2. 必须遵守的产品事实

### 2.1 AI 的角色边界
- AI 不代替法师弘法与开示
- AI 不能以“权威解释者”身份输出主观结论
- AI 回答主要基于知识库原文内容进行引用、摘录、归纳
- 每个回答都应尽量展示：原文引用、出处、讲次、可打开原文入口
- 视觉上要让用户感受到“这是一套引用系统”，不是普通聊天机器人

### 2.2 风格边界
- 整站必须是极简风格
- 不要做电商卡片流、资讯门户感、科技产品感
- 不要使用复杂渐变、玻璃拟物、荧光高饱和、高对比炫光
- 不要依赖大面积装饰图、宗教插画、厚重背景图制造氛围
- 可以有极淡的纸本纹理、轻雾渐层、低存在感水印，但都只能作为背景材质

## 3. 视觉方向

### 3.1 主视觉气质
- 像整理良好的讲记与纸本，而不是 App 营销页
- 页面要给人“先静下来，再开始修学”的感觉
- 用秩序、留白、字重、细线和材质感建立高级感

### 3.2 建议色彩
- 背景底色：#F7F3EC
- 次级背景：#EEE7DD
- 卡片背景：#FFFCF7
- 主文字：#2A241F
- 次文字：#7C7067
- 强调色：#C86E4A
- 强调浅层：rgba(200, 110, 74, 0.12)
- 状态辅助：松青、金砂，仅小范围使用

### 3.3 字体建议
- 中文正文：Noto Sans SC
- 数字与英文：DM Sans
- 引文 / 章节标题：Noto Serif SC

字体要求：
- 正文稳、清楚、长时间阅读不累
- 标题略有书卷感，但不能古风海报化

## 4. 页面任务
请优先设计以下页面，全部按移动端视觉稿输出：

1. 首页
2. 分类 / 专辑页
3. 展开播放器
4. AI 问法页
5. 文库首页
6. 我的修学面板

## 5. 每页设计要求

### 5.1 首页
目标：让用户一进入就知道下一步该做什么

必须包含：
- 今日修学 Hero
- 继续听入口
- AI 推荐问题入口
- 文库联动入口
- 极简底部导航

首页首屏原则：
- 第一视觉重点是“继续听”
- 第二重点是“今天适合问什么 / 看什么”
- 不要把首页做成杂乱入口集合

### 5.2 分类 / 专辑页
目标：像书架与目录，不像普通列表

必须包含：
- 系列简介
- 当前进度
- 总讲数 / 推荐下一讲
- 当前播放态
- 集数目录或相关讲记入口

### 5.3 展开播放器
目标：像修学工具条，而不是音乐播放器

必须突出三件事：
- 当前内容
- 播放进度
- 下一步动作

二级工具：
- 睡眠
- 语速
- 循环

不要让这些二级工具压过主内容

### 5.4 AI 问法页
目标：像“引用型法问工作台”，不是聊天应用

回答区结构建议：
- 问题
- 简短归纳
- 原文引用块
- 出处信息
- 打开原文 CTA
- 继续追问建议

重点要求：
- 回答主体应以原文引用为中心
- 归纳只能作为前置摘要，不可压过引用内容
- 引用块像翻开的讲记页，有清晰出处层级
- 欢迎区要明确写出：AI 回答仅供参考，请以法师原始开示为准

### 5.5 文库首页
目标：像讲记索引页，适合继续阅读与检索

必须包含：
- 最近阅读
- 搜索入口
- 系列目录 / 经典目录
- 关键词命中与上下文感

### 5.6 我的修学面板
目标：从设置页升级为修学状态面板

必须包含：
- 用户信息
- 修学寄语
- 连续修学天数
- 听经时长
- 最近阅读 / 历史 / 功课 / 共修入口

## 6. 按钮系统统一标准

请在 Figma 中建立统一按钮组件，不允许每页单独发明按钮。

按钮分为四类：

### 6.1 主按钮
- 用于当前页面唯一最关键动作
- 高度：44px
- 形态：圆角矩形
- 风格：朱砂赤陶填充，白字
- 示例：继续听、继续阅读、发送

### 6.2 次按钮
- 用于可见但不抢主权重的动作
- 高度：44px
- 风格：浅纸色背景或细边框
- 示例：查看原文、进入详情

### 6.3 幽灵按钮
- 用于轻操作、取消、辅助跳转
- 高度：44px
- 风格：透明背景 + 文字

### 6.4 图标按钮
- 统一为 44px 触控容器
- 默认圆形
- 图标居中
- 仅用于搜索、关闭、分享、收藏、更多、切换等工具动作

统一要求：
- 所有按钮共享同一圆角系统、字重系统、反馈节奏
- 文案控制在 2 到 4 个字为主
- 不要出现同页多个等权重主按钮

## 7. 图标系统重新设计要求

请重绘一套极简线性图标，不直接依赖杂糅现成图标。

### 7.1 风格规则
- 画板：24 x 24
- 描边：1.5px 到 1.75px
- 端点与拐角：round
- 语言：温润、克制、可识别
- 不用双色、渐变、实心大面积填充、拟物细节

### 7.2 需要优先重绘的图标
- 首页
- 听经 / 分类
- 播放 / 暂停 / 上一讲 / 下一讲
- AI 问法
- 文库
- 我的
- 搜索
- 分享
- 收藏
- 下载
- 定时
- 列表
- 关闭

### 7.3 图标尺寸规范
- 20px：列表内辅助图标
- 24px：默认图标
- 28px：重点入口图标
- 但点击区域统一按 44px 容器处理

## 10. 一句话总提示词
为一个佛教净土修学移动网站设计极简、安静、纸本书卷气的 UI 系统。AI 页面不是聊天机器人，而是基于知识库原文引用的问法工作台。整体必须克制、统一、可长期使用，按钮和图标系统要完全一致，避免科技感、电商感和装饰堆砌。
